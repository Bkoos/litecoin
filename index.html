Welcome to THCoins' page

I've created this project to shed som light on the current state of cannabis research. This means that I will donate as much as possible to research in that field. So I've premined about 77 blocks just for this purpose. This is of course, if the coin gets any value.

Coin Specs

  • Scrypt
  • 100 coins per block
  • ~35M coins
  • 3 minute block targets
  • subsidy halves in 350k blocks (~2 years)
  • 720 blocks to retarget difficulty
